3. Began a new website.

4. Wrote some Ruby in conjunction with [Jekyll](https://github.com/mojombo/jekyll).

5. Learned some [Rust](http://www.rust-lang.org/) and wrote a [small irc bot](https://github.com/treeman/rustbot) with it.
