Hey everyone!

This is the list of all the changes we've done to our projects during the month of January.

Summary Of Changes
----------

* **[100r.ca](https://100r.ca)**, added [galley refit](https://100r.ca/site/galley_refit.html), updated [lifelines](https://100r.ca/site/lifelines.html#dec2023), [seamed box cushions](https://100r.ca/site/making_seamed_box_cushions.html#trapezoid), and [moisture prevention underliner](https://100r.ca/site/moisture_prevention_underliner.html).
* **[Permacomputing](http://wiki.xxiivv.com/site/permacomputing.html)**, revamped the page.
* **[Flappy Bird](https://git.sr.ht/~rabbits/uxn-games/tree/main/item/flap/flap.tal)**, released an [Uxn](https://100r.ca/site/uxn.html) version of the Flappy Bird game.
* [Grimgrains](https://100r.ca/site/grimgrains.html), released a new basic recipe: [Whole Wheat Pancakes](https://grimgrains.com/site/whole_wheat_pancakes.html).
* **[Aliceffekt](https://soundcloud.com/neauoire)**, Devine wrote some new music.

On January 10th the forge that we use to host our projects was taken down by DDoS attacks and was struggling to come back online(it's back now, read the [post-mortem](https://sourcehut.org/blog/2024-01-19-outage-post-mortem/)), the event reminded us that we ought to host mirrors and release versions of these source files ourselves. We have begun to [host copies](http://wiki.xxiivv.com/etc/) across our various websites. The builds are still accessible through [itch.io](https://hundredrabbits.itch.io/). These will be automatically updated as we work on them in their individual repositories, but mirrored there for reliability. We are thankful for Sourcehut's tireless work on resolving the issue and for taking the time to communicate important changes.

In keeping with the spirit of improving the resilience of the tools we use we've taken a moment to write a kind of pocket version of the console emulator and [self-hosted assembler](http://wiki.xxiivv.com/site/drifblim.html) as to see how many lines are needed to start from the seed assembler and replicate it. A copy of the pocket emulator, the source for the assembler and its hexadecimal representation have been [documented](http://wiki.xxiivv.com/site/uxn.html).

On January 17th [Victoria](https://100r.ca/site/victoria.html) got its first [snowfall](https://100r.ca/media/content/stream/2024_snow.jpg), with it came temperatures below freezing. We got to test our recent improvements, like a new [louvered closet vent](https://100r.ca/media/content/boat_projects/vent.jpg) to help ventilate the space(there are also two existing vents at the top, one on each side). The closet has been dry for the first time in 3 years. We've made an effort not to keep too many items on the floor so the area can breathe. We got ice inside of the windows for the first time ever though... not ideal.

See Uxn running on a [Zaurus Husky](https://loutre.info/@Hiro/111784989319495175)(Mastodon).
