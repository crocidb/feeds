The GitHub CLI is been a major time saver for me. I was skeptical about its usage in the beginning but it proved to be very useful. Cloning When I want to download a new repository, instead of typing the whole URL or going to the browser to find the repository, I just type: gh repo clone name-of-the-repository It works out of the box if the repo is mine.
