Book: Guia Front-End: O caminho das pedras para ser um dev Front-End by Diego Eis. Rating: ⭐️⭐️⭐️⭐️. O Diego Eis está aí desde a época que usávamos tabelas para construir sites, é sempre interessante absorver conteúdo de pessoas com tanto histórico. O livro não foi escrito para o meu perfil, mas mesmo assim me fez pensar em umas coisas interessantes. Ele me fez realizar o mundo sem CSS para criação de páginas, onde você precisava colocar o estilo em todas as tags, sendo que as tags que estão dentro dela não herdariam o estilo.
