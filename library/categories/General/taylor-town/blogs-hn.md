hello world,

[blogs.hn](https://blogs.hn) is a directory of tech sites, primarily sourced from [HackerNews](https://news.ycombinator.com).

To submit/update a blog, [edit blogs.json in a pull-request](https://github.com/surprisetalk/blogs.hn/blob/main/blogs.json).

If you like sites with RSS feeds, consider checking out [ooh.directory](https://ooh.directory). If you don't have an RSS reader, I highly recommend [Reeder 5](https://reederapp.com).

And if you aren't already, *you should* [write things on the internet](https://www.benkuhn.net/writing/). All you need is [worstpress](https://github.com/surprisetalk/worstpress) and some markdown files! You can [email me](mailto:hello@taylor.town) if you need help getting started.

Every blog is a window into a skull. Don't be afraid to ask questions and kindle friendships! Remember to be kind, courteous, and succinct.
