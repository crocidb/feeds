I plan to learn Spanish, Japanese, German, and Chinese over the next decade.

Here's my current gameplan for each language:

* Listen to Paul Noble's audio courses (\~12 hours total).
* Add beginner news/chat podcasts to my feed (\~50 hours).
* Install an auto-translate browser extension (\~100 hours).
* Consume beloved media in target language (*without* English subtitles) (\~100 hours).
* Practice by writing short stories and speaking with strangers (\~200 hours).

I'm unsure if this plan will work, so please [email me](mailto:hello@taylor.town?subject=Languages) if you have any tips or resources!
