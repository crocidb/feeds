[Markdown](http://daringfireball.net/projects/markdown?utm_source=taoofmac.com&utm_medium=web&utm_campaign=unsolicited_traffic&utm_content=external_link) is a text-to-HTML idiom, conversion tool and all-out ecosystem for web writers, content editors and… [AI](/space/ai), allowing you to write using an easy-to-read, easy-to-write plain text format, then convert it to structurally valid (X)HTML.

Althought I used to prefer [Textile](/space/markup/textile) for its table support and other niceties, Markdown has won that war and enjoys indisputably wider adoption, as well as making for slightly more readable text when writing essays.

Amusingly, it became the *lingua franca* of [LLMs](/space/ai/llm) and is now the de facto standard for most modern documentation as well.
